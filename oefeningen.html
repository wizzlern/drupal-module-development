Oefeningen voor de Drupal Training Day workshop Programmeren.

Inhoud aanmaken

Als redactie van de website wil ik zelfstandig pagina’s kunnen aanmaken en bewerken.

  • Log-in als beheerder.
  • Maak een artikel aan (Navigatie: Manage > Content > Add content > Article).
  • Voer in: ‘title’, ‘body’, ‘tags’ en ‘image’.
  • Sla het artikel op (Knop ‘Save and publish’)
  • Bekijk het resultaat.

Inhoudstypes

Als redactie kan ik game-recenties invoeren. Iedere game kan worden voorzien van een leeftijdscategorie.

  • Maak een inhoudstype ‘game’ aan.
  • Voeg een veld toe voor de leeftijdscategorie. 
    • Naam: field_allowed_age
    • Type: List (integer)
    • Allowed values: 3, 7, 12, 16, 18
  • Maak een paar game-recenties aan.

 

Als site eigenaar wil ik dat gebruikers alleen game-recenties kunnen zien van games die met hun leeftijd overeenkomen.

  • Voeg aan een gebruikersprofiel een veld toe voor de leeftijd. 
    • Naam: field_user_age
    • Type: Number (integer)
    • Required
  • Maak een nieuwe gebruiker aan.

Regions en blokken

Als site eigenaar wil graag laten zien dat mijn site met Drupal gemaakt is.

  • Plaats het blok ‘Powered by Drupal’ in de voet van de pagina. (Navigatie: Manage > Structure > Blocks).
  • Controleer het resultaat.

Drupal module

Leg de basis voor de wizzlern_pegi-module.

  • Kopieer de 'wizzlern_pegi' module uit het oefeningen ‘1 module’ naar /modules/custom.
  • Bekijk de inhoud van wizzlern_pegi.info.yml.
  • Schakel de module in.

Formulier aanpassen

Verwijder de wacthwoord-helptekst van het inlogformulier.

  • Bepaal het form_id van het te wijzingen formulier.
  • Kopieer het wizzlern_pegi.module bestand uit oefeningen ‘2 forms’. Maak de oefeningen voor het inlogformulier.

 

Plaats het leeftijdveld onder het veld ‘Email address’

  • Bepaal het form_id van het te wijzigen formulier.
  • Maak de oefeningen voor het user registration-formulier.

Een blok

Maak het blok waar straks de lijst met games in komt.

  • Kopieer het bestand NewGames.php uit oefeningen ‘3 block’.
  • Maak de oefeningen in dit bestand
  • Plaats het blok op de voorpagina.

Entities

Toon een lijst met games in het ‘New games’-blok.

  • Kopieer het bestand NewGames.php uit oefeningen ‘4 entity’ en overschrijf je eigen NewGames.php bestand.
  • Maak de oefeningen in dit nieuwe bestand.

Access control

Beperk de toegang tot game-recenties op basis van leeftijd van de (ingelogde) gebruiker.

  • In hook_node_access() wordt gecontroleerd of de gebruiker toegang heeft tot de node.
  • Kopieer de functie wizzlern_pegi_node_access() uit wizzlern_pegi.module uit oefeningen ‘5 access control’.
  • Maak de oefeningen die je daarin vind.
  • Pas de toegangscontrole toe op de lijst met games in het blok. Kopieer de NewGames::build() method uit NewGames.php uit oefeningen ‘5 access control’.
  • Maak de oefeningen die je daarin vind.

 

Pas de toegangscontrole toe op de lijst met games in het blok.

  • Vergelijk de code van NewGames::build() uit oefeningen ‘5 access control’ met je eigen code.
  • Kopieer de oefeningen uit NewGames::build() naar je eigen code.
  • Maak deze oefeningen.

Een pagina

Maak een pagina op URL ‘games’ met een lijst van games. Gebruik de ‘teaser’ view-mode voor weergave.

  • Maak een route-definitie. Kopieer wizzlern_pegi.routing.yml uit oefeningen ‘5 pagina’.
  • Maak de oefeningen die je daarin vind.
  • Maak een content controller class met een callback method. Kopieer WizzlernPegiContoller.php uit oefeningen ‘5 pagina’.
  • Maak de oefeningen die je daarin vind.
